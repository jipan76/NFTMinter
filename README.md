# BASE NFT Minter Dapp Example

This is an NFT minter dapp on Base.

## Tech Stack Used

- React
- Ethers
- Hardhat
- Solidity
- Hardhat
- Pinata
- Material UI

## Features

- Connect Wallet & See Wallet address & balance
- Upload Images/Jpeg/Videos directly to IPFS before minting
- Mint on Base
- Intuitive UI statuses for minting
